# Phase 3 IDE Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the `apps/ide` React shell that wires a Blockly workspace to a read-only text panel, a Run button, and an HTML canvas turtle stage so that all 5 example programs run end-to-end in the browser.

**Architecture:** Vite + React 19 single-page app; `BlockWorkspace` owns the Blockly DOM and exposes text changes via a callback; `App` holds all state (`programText`, `canvasCommands`, `error`) and passes slices down to read-only `TextPanel` and `Stage`; the Run button compiles the workspace with `compileWorkspace → interpret → render` and triggers a canvas repaint.

**Tech Stack:** React 19, Vite 6, Blockly (browser), `@sprout/blocks` (registerAllBlocks, compileWorkspace, generateText), `@sprout/lang` (interpret, render, CanvasCommand)

`@sprout/blocks` imports from `blockly/node`. In a browser Vite bundle Vite resolves this through the package's `exports` map, but it is safest to alias it to the main browser entry so both the IDE and `@sprout/blocks` share one Blockly instance (and therefore the same `Blockly.Blocks` registry).

```ts

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      'blockly/node': 'blockly',
    },
  },
});
```

- [ ] **Step 2: Add `blockly` as a direct dep in `apps/ide/package.json`**

> If `workspace:*` fails (blockly is not a local package), use `"blockly": "*"` instead, then run `bun install` from the repo root.

### Task 3: `BlockWorkspace` Component

**Files:**
- Create: `apps/ide/src/BlockWorkspace.tsx`

The component injects a Blockly workspace into a div via a React ref, registers all Sprout blocks, attaches a change listener, and cleans up on unmount.

- [ ] **Step 1: Create `apps/ide/src/BlockWorkspace.tsx`**

```tsx
import { useEffect, useRef } from 'react';
import * as Blockly from 'blockly';
import { registerAllBlocks, generateText } from '@sprout/blocks';

const TOOLBOX: Blockly.utils.toolbox.ToolboxDefinition = {
  kind: 'flyoutToolbox',
  contents: [
    { kind: 'block', type: 'sprout_forward' },
    { kind: 'block', type: 'sprout_turn' },
    { kind: 'block', type: 'sprout_pen_up' },
    { kind: 'block', type: 'sprout_pen_down' },
    { kind: 'block', type: 'sprout_repeat' },
    { kind: 'block', type: 'sprout_def' },
    { kind: 'block', type: 'sprout_call_stmt' },
    { kind: 'block', type: 'sprout_beside' },
    { kind: 'block', type: 'sprout_above' },
    { kind: 'block', type: 'sprout_scale' },
    { kind: 'block', type: 'sprout_on_event' },
    { kind: 'block', type: 'sprout_number' },
    { kind: 'block', type: 'sprout_ident' },
    { kind: 'block', type: 'sprout_infix' },
    { kind: 'block', type: 'sprout_call_expr' },
  ],
};

interface Props {
  onTextChange: (text: string) => void;
  onWorkspaceReady: (ws: Blockly.Workspace) => void;
}

export function BlockWorkspace({ onTextChange, onWorkspaceReady }: Props) {
  const divRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    registerAllBlocks();
    const ws = Blockly.inject(divRef.current!, { toolbox: TOOLBOX });
    onWorkspaceReady(ws);

    const listener = (e: Blockly.Events.Abstract) => {
      if (e.isUiEvent) return;
      onTextChange(generateText(ws));
    };
    ws.addChangeListener(listener);

    return () => {
      ws.removeChangeListener(listener);
      ws.dispose();
    };
    // Callbacks are stable refs from App — intentional empty dep array.
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, []);

  return <div ref={divRef} style={{ width: '100%', height: '100%' }} />;
}
```

### Task 5: `Stage` Component

**Files:**
- Create: `apps/ide/src/Stage.tsx`

The stage is an HTML canvas 500×500 px. It repaints whenever the `commands` prop changes. The Sprout renderer starts the turtle at origin `(0, 0)` heading north, so we translate by `(250, 250)` (canvas center) when drawing.

- [ ] **Step 1: Create `apps/ide/src/Stage.tsx`**

```tsx
import { useEffect, useRef } from 'react';
import type { CanvasCommand } from '@sprout/lang';

const W = 500;
const H = 500;

interface Props {
  commands: CanvasCommand[];
}

export function Stage({ commands }: Props) {
  const canvasRef = useRef<HTMLCanvasElement>(null);

  useEffect(() => {
    const canvas = canvasRef.current!;
    const ctx = canvas.getContext('2d')!;
    ctx.clearRect(0, 0, W, H);
    if (commands.length === 0) return;

    ctx.strokeStyle = '#2563eb';
    ctx.lineWidth = 2;
    ctx.lineCap = 'round';
    ctx.lineJoin = 'round';
    ctx.beginPath();
    // Turtle origin is (0,0); map to canvas center.
    ctx.moveTo(W / 2, H / 2);

    for (const cmd of commands) {
      switch (cmd.kind) {
        case 'moveTo':
          ctx.moveTo(W / 2 + cmd.x, H / 2 + cmd.y);
          break;
        case 'lineTo':
          ctx.lineTo(W / 2 + cmd.x, H / 2 + cmd.y);
          break;
        case 'penDown':
        case 'penUp':
          // moveTo/lineTo already encode pen state; these commands are no-ops for canvas.
          break;
      }
    }
    ctx.stroke();
  }, [commands]);

  return (
    <canvas
      ref={canvasRef}
      width={W}
      height={H}
      style={{
        border: '1px solid #e2e8f0',
        borderRadius: 4,
        background: '#fff',
        display: 'block',
      }}
    />
  );
}
```

### Task 6: Full `App.tsx` — Run Button + Interpret/Render Wiring

**Files:**
- Modify: `apps/ide/src/App.tsx`

This replaces the stub with the production `App`. The Run button calls `compileWorkspace → interpret → render` and updates `commands` state which triggers Stage repaint.

- [ ] **Step 1: Rewrite `apps/ide/src/App.tsx`**

```tsx
import { useRef, useState } from 'react';
import * as Blockly from 'blockly';
import { compileWorkspace } from '@sprout/blocks';
import { interpret, render, SproutRuntimeError } from '@sprout/lang';
import type { CanvasCommand } from '@sprout/lang';
import { BlockWorkspace } from './BlockWorkspace';
import { TextPanel } from './TextPanel';
import { Stage } from './Stage';

export function App() {
  const wsRef = useRef<Blockly.Workspace | null>(null);
  const [programText, setProgramText] = useState('');
  const [commands, setCommands] = useState<CanvasCommand[]>([]);
  const [error, setError] = useState<string | null>(null);

  function handleRun() {
    const ws = wsRef.current;
    if (!ws) return;
    try {
      setError(null);
      const program = compileWorkspace(ws);
      const drawing = interpret(program);
      setCommands(render(drawing));
    } catch (e) {
      setError(e instanceof SproutRuntimeError ? e.message : String(e));
      setCommands([]);
    }
  }

  return (
    <div style={{ display: 'flex', height: '100%', fontFamily: 'sans-serif' }}>
      {/* Left: Blockly workspace */}
      <div style={{ flex: '1 1 0', minWidth: 0 }}>
        <BlockWorkspace
          onTextChange={setProgramText}
          onWorkspaceReady={ws => { wsRef.current = ws; }}
        />
      </div>

      {/* Right: controls + text panel + stage */}
      <div
        style={{
          width: 524,
          display: 'flex',
          flexDirection: 'column',
          padding: 8,
          gap: 8,
          background: '#f8fafc',
          borderLeft: '1px solid #e2e8f0',
        }}
      >
        <button
          onClick={handleRun}
          style={{
            padding: '8px 0',
            fontSize: 15,
            fontWeight: 600,
            background: '#2563eb',
            color: '#fff',
            border: 'none',
            borderRadius: 6,
            cursor: 'pointer',
          }}
        >
          ▶ Run
        </button>

        <TextPanel text={programText} />

        <Stage commands={commands} />

        {error && (
          <pre
            style={{
              background: '#fef2f2',
              color: '#dc2626',
              padding: 8,
              borderRadius: 4,
              fontSize: 12,
              margin: 0,
              whiteSpace: 'pre-wrap',
            }}
          >
            {error}
          </pre>
        )}
      </div>
    </div>
  );
}
```
