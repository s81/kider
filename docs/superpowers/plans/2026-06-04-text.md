# Canvas Text Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `text(str, size)` as a Drawing primitive that renders a string at the turtle's left baseline in the current pen color.

**Architecture:** Follows the same pipeline as the other shape primitives (circle/rect/ellipse/triangle/polygon): Drawing variant in values.ts → CanvasCommand in renderer.ts → drawUpTo case in stage-utils.ts → Blockly block + compiler. The Blockly block uses an inline text field for the string (typed directly in the block) rather than a value input, since Sprout has no string-value block.

**Tech Stack:** TypeScript, Vitest, Blockly 10, pnpm monorepo (`@sprout/lang`, `@sprout/blocks`, `@sprout/ide`)
